# Sanchari – Overview

Sanchari is a React Native travel app that helps users discover, create, and manage trips and experiences. It supports multiple user roles (Traveler, Host, Creator, etc.), KYC verification flows, and Firebase-backed authentication & data.

## Purpose and Target Audience
- Travelers seeking curated trips and experiences
- Creators and Hosts who publish content, experiences, and itineraries
- Admins verifying user credentials and managing approvals

## Key Features
- Secure sign-in with Firebase Auth
- KYC flows with document upload and verification
- Role upgrades (e.g., Traveler to Host/Creator)
- Create posts, reels, packages, stays, rides, and more
- Explore and manage trips

## Screens and Flows
- Auth Flow: Login, Password Changed, Onboarding
- Dashboard & Home
- KYC: Account Change Flow, Host/Creator/StayHost verification
- Create: Post, Reel, Package, and more
